Namespace Commons.Server.Scripts

Classes

ScriptClan

Class that corresponds to one clan

ScriptColor

Color class

ScriptDropItem

Class corresponding to the item dropped in the field

ScriptEnemyUnitAI

Monster AI Control Class

ScriptEventPublisher

This is a class that allows functions in Lua scripts to be used for event registration.

ScriptField

The object that manages the map on the server.
Corresponds to one map

ScriptObject

This is the base class for scripting objects.
Custom data can be modified (custom variables such as ai.customData.test = 1 can be used during scripting of Monster AI)

ScriptParty

This class is for one party. Used to get information about the party or to manipulate the party

ScriptPetUnit

Script class representing the pet.

ScriptPetUnitAI

This is a class to access and control the pet's AI.

ScriptPoint

A class that holds information about points

ScriptRect

Class that saves zone information

ScriptRoomPlayer

The unit for managing game users on the server. It corresponds to one user.

ScriptServer

A server class that allows game servers to be scripted.
Event handlers (listeners) can be added to in-game events such as user access, disconnection, and chat in the game.

ScriptUnit

Unit class, which holds unit information.

ScriptUnitBuff

Class that corresponds to one unit buff

ScriptUtility

A script class with several utility functions.
