GetItems()

The items that this player has, e.g. Titem when using the script below when the event is clicked print (unit.player.GetItems () [1])

Declaration
public List<TItem> GetItems()
Returns
Type Description
System.Collections.Generic.List<TItem>

TItem table
Examples
-- Get a list of the player's inventory items and print them all

local invenItems = unit.player.GetItems()

-- Handling when the warehouse is empty
if #invenItems <= 0 then
    Server.SendSay(unit.name .. "'s inventory is empty")
    return
end

Server.SendSay(unit.name .. "'s inventory items list")

for i = 1, #invenItems do
    if invenItems[i] ~= nil then
        local titem = invenItems[i]
        local itemData = Server.GetItem(titem.dataID)
        
        if itemData ~= nil and titem.count ~= 0 then
            Server.SendSay(i .. ' : ' .. itemData.name .. ' ' .. titem.count .. '개')
        end
    end
end
