What is a script

At Nekoland Studio, you can use Lua scripts to manipulate the game's interface, handle multiple events, and more.

There are two types of scripts: Server script and Client script.

Normally, server scripts are responsible for handling user events in the game, and client scripts are responsible for user interface manipulation.

Getting Started

You can also write a Lua script using a simple editor like Notepad,

You can write scripts more conveniently using editors such as Notepad++ and Visual Studio Code.

Download Notepad++: https://notepad-plus-plus.org/downloads/

Download Visual Studio Code: https://code.visualstudio.com/

Scripting Rules

All script files must end with *.lua extension.

- If it is a client script file, it must exist in the /Scripts folder in the project folder.

- If it is a server script file, it must exist in the /ServerScripts folder in the project folder.
