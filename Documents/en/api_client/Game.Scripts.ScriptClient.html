Class ScriptClient

A class that allows game clients to manipulate scripts.
You can add event handlers (listeners) to multiple user input events in the game.

-- deltaTime represents the time taken to complete the last frame.

        oneSecond = oneSecond + deltaTime
        if oneSecond >= 1 then 
            oneSecond = 0
            print("oneSecond")
        end
    end)

Parameters
Type Name Description
System.String topic

Topic id to send
Returns
Type Description
ScriptEventPublisher
Examples
-- Outputs the content ("Event") when a FIreEvent is received from the server
-- ex) server.lua -> Server.FireEvent("Fire","Event")
Client.GetTopic("Fire").Add(
function(text) 
    print(text)
end)

-- "Event", 1 output
-- ex) server.lua -> Server.FireEvent("Fire2","Event",1)
Client.GetTopic("Fire2").Add(
function(text,index) 
    print(text)
    print(index)
end)
