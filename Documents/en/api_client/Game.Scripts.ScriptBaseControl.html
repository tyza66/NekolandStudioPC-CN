Class ScriptBaseControl

This is the base class for all GUI controls displayed on the screen.

Methods

DOKill()

End all animations that this control is running.

Declaration
public void DOKill()
Examples

-- Create panel controls and move, rotate, and resize them.
-- After 0.5 seconds, all actions are canceled using DOKill.
local panel = Panel(Rect(100, 100, 100, 100))

-- Transform control
panel.DOMove(Point(100, 200), 1);
panel.DOScale(Point(2, 2), 1);
panel.DORotate(180, 1);

Client.RunLater(
    function()
        panel.DOKill();

    end, 0.5);
