Class ScriptButton

Script button class on the screen GUI. You can detect user clicks and add handlers for click events.

Examples
-- Button text control
btn = Button("OK",Rect(100,100 ,100,100))
btn.text = "OK2"

-- Center text
btn.textAlign = 4

-- Resize text
btn.textSize = 15

-- Change text color (RED)
btn.textColor = Color(255,0,0) 

-- Change button color (Yellow)
btn.color = Color(255,255,0)
