Class ScriptGridPanel

This is a GUI class that displays child controls in a grid.

Inheritance
System.Object
ScriptBaseControl
ScriptControl
ScriptGridPanel
Namespace: Game.Scripts
Assembly: Assembly-CSharp.dll
Syntax

Examples
-- After creating the grid panel
-- Put an image in a panel
img = Image("Icons/08.png",Rect(0,0,100,100))
img2 = Image("Icons/08.png",Rect(0,0,100,100))

gridPanel = GridPanel()
gridPanel.cellSize = Point(100,100)

-- If horizontal,vertical is set to true, the size of the panel is determined by the size of the elements inside.
gridPanel.horizontal = true
gridPanel.vertical = true
gridPanel.AddChild(img)
gridPanel.AddChild(img2)

gridPanel.pivotX = 0.5
gridPanel.pivotY = 0.5
gridPanel.anchor = 4
gridPanel.x = 0
gridPanel.y = 0
