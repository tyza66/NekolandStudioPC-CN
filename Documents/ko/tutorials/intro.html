스크립트란

네코랜드 스튜디오에서는 Lua 스크립트를 사용해 게임의 인터페이스를 조작하거나, 여러 이벤트들을 처리하는 등의 행동을 할 수 있습니다.

스크립트의 종류는 크게 2가지로 서버 스크립트와 클라이언트 스크립트로 나뉘는데

보통 서버 스크립트에서는 게임 내의 유저 이벤트 처리를 담당하고, 클라이언트 스크립트에서는 유저 인터페이스 조작을 담당합니다.

시작하기

Lua 스크립트를 작성할 때 메모장처럼 간단한 에디터를 사용해 작성할 수도 있지만,

Notepad++ 와 Visual Studio Code 등의 에디터를 사용하면 더 편리하게 스크립트를 작성할 수 있습니다.

Notepad++ 다운로드: https://notepad-plus-plus.org/downloads/

Visual Studio Code 다운로드: https://code.visualstudio.com/

스크립팅 규칙

모든 스크립트 파일들은 *.lua 의 확장자로 끝나야 합니다.

- 클라이언트 스크립트 파일이라면 프로젝트 폴더 내의 /Scripts 폴더 내에 존재해야 합니다.

- 서버 스크립트 파일이라면 프로젝트 폴더 내의 /ServerScripts 폴더 내에 존재해야 합니다.
