예제
-- 내주변 200거리안에서 가장 큰 hp를 가진 적유닛을 가져온 후 이름을 출력한다.
local me = Client.myPlayerUnit
local findUnit = Client.field.FindMaximumUnit(me.x, me.y, 200,
    function(unit)
        return unit.hp
    end
, 2, me)

if findUnit ~= nil then
    print(findUnit.name)
end

찾는 거리 범위

유닛 검색 콜백 함수 (인자로 해당 유닛 객체를 받고, bool 형식의 반환값으로 찾는 대상인지 판별합니다)
System.Int32 findType

탐색할 유닛 타입 (0: 플레이어, 1: 이벤트 유닛 , 2: 적)

ScriptUnit
예제
-- 내주변 200거리안에서 hp가 100 이하인 적 유닛을 찾고 이름을 출력한다.
- unit -> scriptUnit
local me = Client.myPlayerUnit
local findUnit = Client.field.FindUnit(me.x, me.y, 200,
                       function(unit)
                                return unit.hp <= 100
                        end
                , 2 ,me)
if findUnit ~= nil then
    print(findUnit.name)
end
