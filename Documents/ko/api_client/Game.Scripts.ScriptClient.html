Class ScriptClient

게임 클라이언트를 스크립트로 조작할 수 있게 해주는 클래스입니다.
게임 내에서 일어나는 여러 사용자 입력 이벤트에 이벤트 처리기(리스너)를 추가할 수 있습니다

상속
System.Object
ScriptClient
구현
EventListener
네임스페이스: Game.Scripts
어셈블리: Assembly-CSharp.dll
Syntax
[MoonSharpUserData]
public class ScriptClient : EventListener

타입 이름 설명
System.String topic

보낼 Topic
반환
타입 설명
ScriptEventPublisher
예제
-- 서버의 FIreEvent로 전달받은경우 내용(Event) 출력
-- ex) server.lua -> Server.FireEvent("Fire","Event")
Client.GetTopic("Fire").Add(
function(text) 
    print(text)
end)

-- "Event", 1 출력
-- ex) server.lua -> Server.FireEvent("Fire2","Event",1)
Client.GetTopic("Fire2").Add(
function(text,index) 
    print(text)
    print(index)
end)
