Class ScriptSlider

사용자에게 정해진 범위 내의 숫자를 입력받을 수 있는 슬라이더를 나타내는 GUI 클래스입니다

상속
System.Object
ScriptBaseControl
ScriptControl
ScriptSlider
네임스페이스: Game.Scripts
어셈블리: Assembly-CSharp.dll
Syntax
[MoonSharpUserData]
public class ScriptSlider : ScriptControl
예제
-- 위치(X : 400, Y : 250) 크기(W : 100, H : 50)를 가진 슬라이더를 생성합니다.
slider = Slider(Rect(400, 250, 100, 50))

-- 배경이미지를 변경 할 수 있습니다.
slider.backgroundImage = "Pictures/배경.png" 

-- 채워질 이미지를 생성 할 수 있습니다.
slider.fillImage = "Pictures/배경1.png"
