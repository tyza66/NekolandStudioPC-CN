Class ScriptSprite

스프라이트를 나타내는 그래픽 클래스입니다

상속
System.Object
ScriptSprite
네임스페이스: Game.Scripts
어셈블리: Assembly-CSharp.dll
Syntax
[MoonSharpUserData]
public class ScriptSprite
예제
-- 위치(X : 400, Y : 250) 크기(W : 100, H : 50)를 가진 입력한 경로의 스프라이트를 생성합니다.
-- 스프라이트는 입력 한 크기로 맞춰집니다.

text = Sprite('Icons/A.png', Rect(400, 250, 100, 50))
