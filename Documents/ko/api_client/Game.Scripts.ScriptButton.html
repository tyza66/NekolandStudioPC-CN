Class ScriptButton

화면 GUI 상의 스크립트 버튼 클래스입니다. 사용자의 클릭을 감지하고 클릭 이벤트에 대한 처리기를 추가할 수 있습니다.

상속
System.Object
ScriptBaseControl
ScriptControl
ScriptButton
네임스페이스: Game.Scripts
어셈블리: Assembly-CSharp.dll
Syntax
[MoonSharpUserData]
public class ScriptButton : ScriptControl
예제
-- 버튼 텍스트 조절
btn = Button("OK",Rect(100,100 ,100,100))
btn.text = "OK2"
-- 가운데 정렬
btn.textAlign = 4
-- 텍스트 크기 조절
btn.textSize = 15
-- 텍스트 색 조절 (RED)
btn.textColor = Color(255,0,0) 
-- 버튼 색 조절 (Yellow)
btn.color = Color(255,255,0)
