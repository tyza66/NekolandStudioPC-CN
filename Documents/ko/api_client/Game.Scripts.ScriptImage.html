Class ScriptImage

이미지를 나타내는 GUI 클래스입니다

상속
System.Object
ScriptBaseControl
ScriptControl
ScriptImage
네임스페이스: Game.Scripts
어셈블리: Assembly-CSharp.dll
Syntax
[MoonSharpUserData]
public class ScriptImage : ScriptControl
예제
-- 이미지에 쿨타임 넣기
coolTime = 10
leftTime = 10
img = Image("Icons/08.png",Rect(0,0,100,100))
img.imageType = 3
img.fillMethod = 4
img.fillClockwise = true
img.anchor = 4
img.pivotX = 0.5
img.pivotY = 0.5
img.x = 0
img.y = 0

function CoolTime(deltaTime)
    leftTime = leftTime - deltaTime
    if leftTime <= 0 then
        leftTime = coolTime
        print("end")
        img.fillAmount = 1
         Client.onTick.Remove(CoolTime)
        return
    end
    img.fillAmount = 1 - (leftTime / coolTime)
end

Client.onTick.Add(CoolTime)
