Class ScriptUnit

게임 내 유닛을 나타내는 클래스입니다 (클라이언트)

상속
System.Object
ScriptUnit
ScriptMyPlayerUnit
네임스페이스: Game.Scripts
어셈블리: Assembly-CSharp.dll
Syntax
[MoonSharpUserData]
public class ScriptUnit
예제
-- 현재 나의 플레이어 유닛 정보를 얻어옵니다.

-- 클라이언트 스크립트에서 다음과 같이 나의 플레이어 유닛을 참조할 수 있습니다.

-- 유닛 ID
print('ID : ' .. Client.myPlayerUnit.id)

-- 이름
print('Name : ' .. Client.myPlayerUnit.name)

-- 위치
print('X : ' .. Client.myPlayerUnit.x .. ' ' .. 'Y : ' .. Client.myPlayerUnit.y)
